What is Git? Why do we need Source Code Control Systemoki

Git is a free software tracking system that is used by developers and teams to track changes in codes. Git is used mainly by teams simultaneoulsy working on coordinating codes and files. Version control is important. Version control is important as codes and files that undergo various revisions and receive various inputs from different users might need to be tracked due to the ease of people being able to change the code. Source code control is also helpful when one needs to find out the version of code that is currently being used amongst the multiple versions. or merge different version of the same code for optimal efficiency.
