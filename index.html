What is Docker

Docker is a software tool that runs applications using containers. Containers are used in order to ensure that application codes are all standardized and thus can be used on any Linux machine even if there are certain settings that are different. Furthermore, Dockers are important as instead of creating a whole virtual host system, they merely provide the necessary applications that are unavailable on the host Linux system. Lastly, Docker is cuztomizable for developers who need to utilize applications that arent available on the standard application containers. Sources - https://aws.amazon.com/docker/ - https://opensource.com/resources/what-docker

What is Nginx

Nginx is a high performance web/proxy server popular for its adapatability, features and simple configuration. Nginx is used by a wide variety of companies for their websites due to it consistently being one of the highest benchmarkers in terms of outperforming other serves for hosting. Nginx employs "asynchronous, event - driven approach" where a master process controls multiple worker processes without blocking other requests thus handling high levels of concurrent requests. What is VM (virtual machine)

A virtual machine is an emulation of a physical computer - the end result is the user being able to receive the same experience on a window of the physical computer as the host operating system. Virtual machines do not affect the processes of the physical operating system and have their own CPU's, netwrks and interfaces. Running VM's is beneficial for testing and employing applications that may not be easily run on the host system. Companies may use VM's due to having a multitude of apps that need to be tested and do not want to spend on physical computers and maintenance/depreciation. Sources - https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/ - https://www.lifewire.com/virtual-machine-4147598

Who is Cloud Computing Service Provider - Digitalocean

Cloud Computing Service Providers are companies that allow users to access network infrastructures, applications or services in the cloud. This means that it can be accessed by anyone with network connectivity. The most popular cloud service is Amazon Web Services. Companies usually offer three different types of cloud computing : 1) IaaS - Infrastructures as a Service 2) SaaS - Software as a Service 3) PaaS - Platform as a Service In particular, Digital Ocean is the thrid largest web facing hosting compnay in the world - they provide services to run apps and deploy on several computers at once.
